Display IP address, user agent, and referrer of the website visitors.

== Description ==
The Visitors plugin allows you to monitor and display information about the visitors to your WordPress site. It captures the IP address, user agent, and referrer of each visitor, providing valuable insights into your site traffic.

= Features =
- Display IP address, user agent, and referrer information.
- Group visitors by date for better analysis.
- Simple and easy-to-use.

== Installation ==
1. Upload the `visitors` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the 'Visitors' menu in the WordPress admin to view visitor information.

== Frequently Asked Questions ==
= How does the plugin capture visitor information? =
The plugin captures the visitor's IP address, user agent, and referrer from the HTTP request headers.

= Can I group visitors by date? =
Yes, the plugin automatically groups visitors by date, making it easier to analyze traffic patterns.
